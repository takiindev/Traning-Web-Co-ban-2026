# 2. PSEUDO-CLASSES (35 PHÚT)

## Nội dung

- `:hover`
- `:focus`
- `:nth-child()`

- `:last-child`
- `:has()`

### Thực hành

- Hover nút.
- Tô màu item thứ 2.
- Làm trạng thái focus cho input.
